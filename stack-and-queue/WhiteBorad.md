## Approach & Efficiency

the big o for this approach will be :

### stack:

- push : Time: O(1) Space: O(1)
- pop : Time: O(1) Space: O(1)
- peek : Time: O(1) Space: O(1)
- is_empty : Time: O(1) Space: O(1)

### Queue:

- enqueue : Time: O(1) Space: O(1)
- dequeue : Time  : O(1) Space: O(1)
- peek : Time: O(1) Space: O(1)
- is_empty : Time: O(1) Space: O(1)
